# 🌸 Iris Exploratory Data Analysis

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on the Iris dataset using Python. It explores the dataset with summary statistics and visualizations.

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab
